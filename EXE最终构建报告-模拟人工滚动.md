## 技术细节

### 滚动算法

```python
# 1. 获取当前位置和页面尺寸
current_position = window.pageYOffset
page_height = document.documentElement.scrollHeight
viewport_height = window.innerHeight

# 2. 计算滚动距离（视口高度的0.5-1倍）
scroll_distance = viewport_height * (0.5 + (n % 3) * 0.2)

# 3. 计算新位置
new_position = min(current_position + scroll_distance, page_height)

# 4. 平滑滚动
window.scrollTo({top: new_position, behavior: 'smooth'})

# 5. 随机等待1-3秒
time.sleep(random.uniform(1.0, 3.0))
```

### 停止条件

```python
# 条件1: 到达底部且无新内容
if at_bottom and no_change_count >= 3:
    stop()

# 条件2: 连续无新内容
if no_change_count >= 6:
    stop()

# 条件3: 达到滚动次数上限
if scroll_attempts >= 100:
    stop()
```
