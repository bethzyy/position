## 🚀 快速开始

### 步骤1: 设置API Key

打开命令提示符(CMD),输入以下命令:

```bash
setx ZHIPU_API_KEY "your-api-key-here"
```

**重要**: 将 `your-api-key-here` 替换为你的实际API Key

### 步骤2: 启动程序

双击运行 `启动工具.bat` 或在命令行输入:

```bash
python main.py
```

### 步骤3: 使用工具

1. **输入评价网址** - 每行一个网址
2. **选择简历文件** - 支持PDF/Word/TXT
3. **粘贴职位描述** - 完整的JD内容
4. **点击"开始分析"** - 等待2-3分钟
5. **查看报告** - 自动打开生成的HTML报告

## 📖 详细使用说明

### 1. 评价网址输入

**支持的网站类型:**
- 看准网 (kanzhun.com)
- 职友集 (jobui.com)
- 脉脉
- 知乎公司评价
- 其他员工评价网站

**注意事项:**
- ✅ 每行一个网址
- ✅ 可以输入多个网址(程序会合并分析)
- ✅ 确保网址完整(包含http://或https://)
- ❌ 不要输入需要登录才能看到的页面

**示例:**
```
https://www.kanzhun.com/pl/company123.html
https://www.jobui.com/company456/reviews
https://www.zhihu.com/question/xxxxx
```

### 2. 简历文件选择

**支持的格式:**
- PDF (.pdf)
- Word文档 (.doc, .docx)
- 纯文本 (.txt)

**简历内容建议:**
- ✅ 包含完整的工作经历
- ✅ 列出掌握的技能栈
- ✅ 突出项目经验
- ✅ 教育背景完整
- ❌ 不要使用扫描版PDF

### 3. 职位描述(JD)输入

**JD内容越详细越好,应包含:**
- 职位名称和级别
- 岗位职责
- 任职要求(技能、经验)
- 加班和出差情况
- 薪资范围
- 公司信息

**示例JD格式:**
```
职位: 高级Python开发工程师
薪资: 25-40K
地点: 北京

岗位职责:
1. 负责后端系统开发
2. 参与系统架构设计
3. 优化系统性能

任职要求:
1. 3-5年Python开发经验
2. 熟练使用Django/Flask
3. 了解微服务架构
4. 有高并发系统经验优先

其他:
- 偶尔需要加班
- 有弹性工作制
- 五险一金+年终奖
```

### 4. 输出报告

**报告内容:**
- **第一部分**: 公司文化分析
  - 公司氛围
  - 员工文化和福利
  - 面试流程
  - 关键发现
  - 建议

- **第二部分**: 职位匹配分析
  - 职位定位
  - 匹配度评估(技能/经验/文化)
  - 职业发展利弊
  - 面试准备清单
  - 最终建议

**报告特点:**
- 📄 精美的HTML格式
- 📱 手机端适配
- 🖨️ 支持打印
- 🔗 包含数据源链接

## ❓ 常见问题

### Q1: 提示"未找到ZHIPU_API_KEY"

**原因**: 未设置环境变量

**解决方案**:
```bash
# 方式1: 临时设置(仅当前窗口有效)
set ZHIPU_API_KEY=your-api-key

# 方式2: 永久设置(推荐)
setx ZHIPU_API_KEY "your-api-key"
```

### Q2: 网页抓取失败

**可能原因:**
1. 网址输入错误
2. 网站有反爬机制
3. 网络连接问题

**解决方案:**
- ✅ 检查网址是否正确
- ✅ 尝试使用其他类似网站
- ✅ 检查网络连接
- ✅ 查看日志了解详细错误

### Q3: AI分析时间很长

**正常情况:**
- 单次AI分析: 30-60秒
- 完整流程: 2-3分钟

**如果超过5分钟:**
- 检查网络连接
- 查看日志文件
- 尝试减少输入网址数量

### Q4: 简历无法识别

**可能原因:**
- 文件格式不支持
- PDF是扫描版
- 文件损坏

**解决方案:**
- ✅ 转换为TXT格式
- ✅ 使用Word格式
- ✅ 确保PDF是文本格式(非图片)

### Q5: 生成的报告无法打开

**解决方案:**
- 使用Chrome/Edge浏览器打开
- 确保文件路径没有中文
- 检查文件是否被占用

## 🔧 高级功能

### 配置文件预设

编辑 `config.json` 可以预设参数:

```json
{
  "urls": "https://example.com/review1\nhttps://example.com/review2",
  "resume_path": "C:\\Users\\YourName\\Resume.pdf",
  "job_description": "固定的JD内容",
  "output_path": "output/报告.html"
}
```

### 日志查看

所有AI调用详情记录在 `logs/` 目录:

- 文件名: `ai_calls_YYYYMMDD.log`
- 内容: 完整的输入输出
- 用途: 调试和审计

### 批处理模式

可以创建多个配置文件,依次运行:

```bash
python main.py --config config_company1.json
python main.py --config config_company2.json
```
