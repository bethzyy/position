#### AI分析能力提升

现在AI会特别关注以下风险信号:

1. **外包风险识别**:
   - 劳务派遣、外包员工、第三方合同
   - 非正式编制（如"项目制"、"灵活用工"）
   - 这些情况会导致稳定性得分大幅降低

2. **变相裁员手段识别**:
   - 强制调岗
   - 恶意绩效压低
   - 不可能的KPI目标
   - 突然降薪/取消福利
   - 部门解散后的无补偿劝退

3. **薪资陷阱识别**:
   - 基本工资极低，绩效占比过高
   - 承诺高额年终奖但难以拿满
   - 五险一金按最低基数缴纳
   - 各种名目的罚款和扣款

## 技术细节

### 超链接实现难点

1. **路径计算**:
   - 问题: JD文件在 `companies/公司名/jd.txt`
   - 报告文件在 `output/公司名.html`
   - 排名报告在 `output/公司推荐排名.html`
   - 解决: 计算正确的相对路径

2. **HTML表格处理**:
   - 问题: AI生成的Markdown表格已转换为HTML
   - 解决: 使用后处理方式，在HTML上直接操作
   - 使用正则表达式定位并替换表格内容

3. **兼容性**:
   - 问题: 需要兼容直接调用 `generate_ranking_report()` 的情况
   - 解决: 添加自动查找JD文件的逻辑

### 正则表达式策略

```python
# 提取公司名称
company_match = re.search(r'<td><strong>(.+?)</strong></td>', row_content)

# 添加JD链接列
row_content = re.sub(r'(</td>\s*</tr>)', f'</td><td>{jd_link}\\1', row_content)

# 表头添加JD列
html = re.sub(r'(<thead><tr>.*?<th>职位匹配度.*?</th>)', r'\1<th>JD链接</th>', html)
```
