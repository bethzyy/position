## 🔧 技术架构

### 技术栈
- **语言**: Python 3.8+
- **GUI**: Tkinter (Python标准库)
- **网页抓取**: Selenium 4.15+
- **AI接口**: Anthropic SDK (GLM-4.6)
- **文档解析**: PyPDF2, python-docx
- **日志**: Python logging

### 关键设计决策

1. **使用GLM-4.6而非GLM-4.7**
   - 原因: GLM-4.7有时返回空内容,稳定性不够
   - 结果: GLM-4.6更加稳定可靠

2. **Anthropic兼容接口**
   - 原因: 统一接口,便于切换模型
   - 结果: 代码更简洁,易于维护

3. **完整日志记录**
   - 记录所有AI调用的元数据、输入、输出
   - 便于调试和审计
   - 日志文件按日期组织

4. **内容长度自适应**
   - 自动截断过长的输入
   - 避免超过token限制
   - 保证分析成功率
