## 修复时间
2026-01-12 14:30

## 问题描述

用户报告运行EXE时出现错误：
```
✗ 分析失败: 提示词文件不存在: prompts\company_culture_analysis.txt
```

### 根本原因

在 `ai_analyzer.py` 中，提示词文件使用相对路径：

```python
prompt_file = Path("prompts/company_culture_analysis.txt")
```

**问题**：
- 当从EXE运行时，`--add-data` 参数指定的数据文件会被PyInstaller解压到临时目录
- 临时目录路径保存在 `sys._MEIPASS` 中
- 使用相对路径会在当前工作目录查找，而不是临时目录
- 导致找不到提示词文件

---

## 修复方案

### 1. 添加资源路径辅助函数

在 `ai_analyzer.py` 顶部添加 `get_resource_path()` 函数：

```python
def get_resource_path(relative_path: str) -> Path:
    """
    获取资源文件的绝对路径（兼容EXE和Python脚本运行）

    Args:
        relative_path: 相对路径

    Returns:
        资源文件的绝对路径
    """
    if getattr(sys, 'frozen', False):
        # 如果是EXE运行，资源文件在临时目录
        # PyInstaller会将--add-data指定的文件解压到sys._MEIPASS
        base_path = Path(sys._MEIPASS)
    else:
        # 如果是Python脚本运行，使用脚本所在目录
        base_path = Path(__file__).parent

    return base_path / relative_path
```

**工作原理**：
- `sys.frozen`: PyInstaller打包的EXE会设置此标志
- `sys._MEIPASS`: EXE运行时的临时资源目录
- `__file__`: Python脚本的文件路径

**解决**：确保所有资源文件都使用 `get_resource_path()`

3. 手动检查路径是否存在

### Q3: sys._MEIPASS是什么？

**说明**：
- PyInstaller在运行EXE时创建的临时目录
- 所有通过 `--add-data` 打包的文件都解压到这里
- 程序退出后会被自动清理

## PyInstaller参数说明

### --add-data

```bash
--add-data "源目录;目标目录"
```

**示例**：
```bash
--add-data "prompts;prompts"           # 打包prompts目录
--add-data "templates;templates"       # 打包templates目录
--add-data "data/data.csv;data"        # 打包单个文件到data目录
```

**注意**：
- Windows使用 `;` 分隔源和目标
- Linux/Mac使用 `:` 分隔源和目标
- 目标路径相对于 `sys._MEIPASS`

## 相关概念

### sys.frozen

```python
if getattr(sys, 'frozen', False):
    # EXE模式
    # sys.frozen = True
    # sys.executable = EXE文件的完整路径
    # sys._MEIPASS = 临时资源目录
else:
    # Python脚本模式
    # sys.frozen 不存在（或为False）
    # __file__ = 脚本文件的完整路径
```

### pathlib.Path

```python
# 路径拼接使用 / 操作符
path = Path("/base") / "subdir" / "file.txt"
# 结果：/base/subdir/file.txt

# 优势：
# - 自动处理不同操作系统的路径分隔符
# - 避免手动拼接字符串
# - 类型安全
```

### 2. 创建资源管理器类

对于大型项目，可以创建专门的资源管理器：

```python
class ResourceManager:
    """资源文件管理器"""

    def __init__(self):
        self.base_path = self._get_base_path()

    def _get_base_path(self) -> Path:
        """获取资源基础路径"""
        if getattr(sys, 'frozen', False):
            return Path(sys._MEIPASS)
        return Path(__file__).parent

    def get_prompt(self, name: str) -> Path:
        """获取提示词文件路径"""
        return self.base_path / "prompts" / name

    def get_template(self, name: str) -> Path:
        """获取模板文件路径"""
        return self.base_path / "templates" / name
```
